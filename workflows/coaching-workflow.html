Coaching Session Workflow

Weekly process for collecting topics, updating session descriptions, and managing paid-member access to coaching.

Topic Collection Session Updates Access Control Paid Members Only
Overview

Two People, Two Jobs

Kate handles execution. Dave reviews and approves. The workflow has two parts: updating the coaching session each week, and locking access to paid members only.

Kate

Execution

Pull topics after deadline, share with Dave, update session description and leadership chat, lock down access.

Dave

Review & Approve

Review submitted topics, confirm what goes into the session description, approve updates before they go live.

Part 1

Weekly Topic Collection & Updates

Every week, members submit coaching topics. After the deadline closes, Kate collects them and works with Dave to update the session.

Submission Deadline

Topic submissions close every week. No late additions.

Tuesday 4:00 PM
1

Pull Submitted Topics

Kate

Once the submission window closes at 4:00 PM Tuesday, collect all topics that were submitted by members.

2

Share Topics with Dave

Kate

Send the compiled topic list to Dave for review. No filtering needed at this stage. Just pass along everything that came in.

3

Review & Confirm Core Topics

Dave

Dave reviews the submitted topics and identifies the core themes for the session. He confirms what goes into the description and flags anything that needs a pre-conversation.

4

Update Coaching Session Description

Kate + Dave

Update the coaching session description with two things:

  • The core topics for the upcoming session
  • A note: "You need to speak with me first" so members know to check in before attending
5

Update the Leadership Chat

Kate

Post the same topic summary and session info into the leadership chat. For now, only focus on this group. Do not worry about other announcements or channels.

Scope note: For now, only update the leadership chat group. No other announcements or channels needed.
Part 2

Lock Down Coaching Access

Coaching stays in the community but is restricted to paid members only. Kate handles the access settings.

Restrict to Paid Members

Kate

Lock the coaching session so only subscribed, paid members can access it. It remains visible in the community, but participation is gated.
